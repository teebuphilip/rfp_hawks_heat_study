Definitions

Teammate-neutral evaluation Player production adjusted to a common roster context.
FMV The cap-constrained market price a player clears for.
Wins number We start with FMV salary, convert dollars to wins with the salary-per-win bridge, apply HBB for team context, then add an offense-first proxy adjustment from the proxy layer. The current proxy layer uses box-score signals only; free-throw attempts are not yet in the feed.
FMVW The wins-equivalent translation of that market value before proxy adjustment.
Proxy-adjusted wins The FMVW win total after the proxy layer is applied. Offense is compressed with a soft cap, and weak defense subtracts more than strong defense adds. The exact weights live in code and can be tuned later.
HBB A team-context multiplier. It measures how much the surrounding roster, balance, pressure, and support change the wins-equivalent value of the player after FMV is set. HBB is not a player trait; it is a team-level adjustment applied on top of player value.
Band A salary tier label derived from FMV, used to sort players into ranges like minimum, rotation, starter, core, and star.
